# A-Share Signal

`a-share-signal` 是一个用于分析A股交易机会的Skill，基于 AkShare 数据与结构化交易框架，对个股进行多维度研判并输出可执行结论。

## 功能简介

- 基于筹码分布、三周期共振、优化 KDJ、威科夫与缠论等框架分析 A 股
- 结合用户自定义评分规则，输出结构化判断、参与条件与应对方案
- 适合把近期行情数据快速转成更清晰的交易决策参考
